## :fire: Como usar

- ### **Pré-requisitos**

  - É **preciso** ter um gerenciador de pacotes seja o **[NPM](https://www.npmjs.com/)** ou **[Yarn](https://yarnpkg.com/)**.

1. Faça um clone :

```sh
  $ git clone https://github.com/williamroberttv/pokeapp.git
```

2. Executando a Aplicação:

```sh
  # Instale as dependências
  $ yarn
  $ npm install

  # Inicie a aplicação
  $ yarn start
  $ npm start

```
